# GitHub Explorer (C++)

A C++ application that fetches and displays GitHub user information using the GitHub REST API.

## Features

* Search GitHub users by username
* HTTP requests using libcurl
* JSON parsing using nlohmann/json
* Display user profile information:

  * Name
  * Username
  * Followers
  * Following
  * Public Repositories
  * Location
  * Profile URL
* Invalid username detection
* Network error handling

## Technologies Used

* C++
* libcurl
* nlohmann/json
* GitHub REST API
* GCC 16 (MSYS2)

## Concepts Practiced

* HTTP Requests
* REST APIs
* JSON Parsing
* Error Handling
* External Libraries
* Data Extraction from APIs
