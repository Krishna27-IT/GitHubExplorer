# GitHub Explorer (C++)

A C++ application that fetches GitHub user information using the GitHub REST API.

## Features

* Search GitHub users by username
* HTTP requests using libcurl
* Dynamic user lookup
* JSON response retrieval from GitHub API

## Technologies Used

* C++
* libcurl
* GitHub REST API
* GCC 16 (MSYS2)

## Current Status

Currently retrieves raw JSON data for any valid GitHub username.

Future improvements:

* JSON parsing with nlohmann/json
* User profile display
* Repository listing
* OOP refactor
* Error handling improvements
